Descripción General

Conoce el sistema, su arquitectura y cómo funciona.

¿Qué es el sistema?

El Sistema de Gestión de Cupos de Viajes Aéreos es una aplicación web diseñada para la gestión corporativa de viajes aéreos. Permite a los usuarios solicitar reservas de vuelos, consultar disponibilidad y gestionar confirmaciones, todo ello con un sistema de seguridad robusto y control de accesos basado en roles.

Estado del sistema: Producción

Todas las vulnerabilidades corregidas • Sistema seguro y estable

Backend

  • • Supabase Edge Functions

Base de Datos

  • • PostgreSQL
  • • Neon / Supabase

Cómo Funciona

1

Login y Autenticación

El usuario ingresa sus credenciales. El sistema verifica el estado de bloqueo, valida las credenciales y si está habilitado, solicita verificación 2FA.

2

Consulta de Disponibilidad

El usuario consulta vuelos disponibles por destino, fecha o agencia. Los datos se obtienen en tiempo real de las fuentes conectadas.

3

Solicitud de Reserva

El usuario selecciona un vuelo y envía una solicitud. Esta pasa por un flujo de aprobación según su rol y permisos.

4

Confirmación y Seguimiento

Una vez aprobada, la reserva se confirma automáticamente. El usuario puede rastrear el estado en cualquier momento desde su dashboard.
